# fst (filesorter)

Bash-утилита для автоматической сортировки файлов по категориям (изображения, видео, документы, архивы) и дате изменения (`ГОД/МЕСЯЦ`).

## Установка

```bash
git clone https://github.com/kingograder/filesorter.git
cd filesorter
chmod +x fst
mv fst ~/.local/bin/
```
*Перезапустите терминал, если `~/.local/bin` отсутствует в `$PATH`.*

## Использование

```bash
fst [ФЛАГИ]
```

| Флаг | Альтернатива | Описание |
| :--- | :--- | :--- |
| `-o` | Целевая директория |
| `-r` | Рекурсивный обход подпапок |
| `-d` | Сортировка по дате изменения |
| `-c` | Копировать вместо перемещения |
| `-a` | Исключения (напр., `"*.tmp"`) |
| `-q` | Тихий режим |
| `-y` | Автосогласие на действия |
| `-h` | Справка |
| `-u` | Сортирует файлы в стандартную директорию пользователя |

## Примеры

**Базовая сортировка текущей папки:**
```bash
fst
```

**Рекурсивная сортировка в домашнюю директорию:**
```bash
fst -o ~/ -r
```
