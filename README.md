# fst - скрипт для сортировки файлов

`fst` (File Sorter) — это утилита командной строки на Bash для автоматической организации файлов в директории по категориям, основанным на их расширениях.

## Возможности

-   **Категоризация:** Сортировка файлов по предопределенным категориям (изображения, видео, документы, архивы и т.д.).
-   **Сортировка по дате:** Создание вложенной структуры `ГОД/МЕСЯЦ` внутри каждой категории на основе даты изменения файла.

## Зависимости

-   **Bash** версии 4.0+ (для поддержки ассоциативных массивов).
-   **xdg-utils** (для определения путей XDG в глобальном режиме). Пакет предустановлен в большинстве современных дистрибутивов.

## Установка

1.  **Загрузите скрипт**
    Клонируйте репозиторий или скачайте файл `fst` напрямую.

    ```bash
    git clone https://адрес_вашего_репозитория.git
    cd fst
    ```

2.  **Сделайте скрипт исполняемым**
    ```bash
    chmod +x fst
    ```

3.  **Переместите скрипт в директорию из вашего `$PATH`**
    Это позволит вызывать скрипт по имени `fst` из любой директории. Рекомендуемый путь для пользовательских скриптов — `~/.local/bin`.
    ```bash
    mv fst ~/.local/bin/
    ```
    *Примечание: Убедитесь, что `~/.local/bin` находится в вашей переменной `PATH`. В большинстве современных систем это настроено по умолчанию. Перезапустите сессию терминала, чтобы изменения вступили в силу.*

## Использование

Скрипт запускается из командной строки с указанием необязательных флагов.

```bash
fst [ФЛАГ]
```

### Флаги

| Флаг короткий | Флаг длинный | Описание                                                                                                |
| :------------ | :----------- | :------------------------------------------------------------------------------------------------------ |
| `-l`          | `--local`    | Сортировать файлы в подпапки, созданные внутри текущей директории (`./Картинки`, `./Видео` и т.д. иначе в /home).        |
| `-r`          | `--recursive`| Включить рекурсивный поиск и обработку файлов во всех вложенных директориях.                            |
| `-d`          | `--date`     | Создавать вложенные папки `ГГГГ/ММ` на основе даты изменения файла.                                  |
| `-c`          | `--copy`     | Копировать файлы вместо перемещения.                              |
| `-q`          | `--quiet`    | Тихий режим.                                                |
| `-v`          | `--verbose`  | Подробный режим.                   |
| `-h`          | `--help`     | Показать справочное сообщение.                                                       |

### Примеры

1.  **Простая сортировка файлов в текущей папке по директориям XDG:**
    ```bash
    fst
    ```

2.  **Рекурсивно переместить все файлы в локальные подпапки (`./Картинки`, `./Архивы`):**
    ```bash
    fst --local --recursive
    ```

3.  **Рекурсивно скопировать все файлы, раскладывая их по дате в стандартные папки (`~/Pictures/2025/11`):**
    ```bash
    fst -r -d -c -l
    ```
